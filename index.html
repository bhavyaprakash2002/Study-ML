What is Machine Learning?

Machine Learning simply means making machines learn. Just like humans, if provided enough data machines can learn certain patterns in the provided data and based on its learning it can produce results which we call as 'Predictions'. Just consider the machines as a small baby who is new to the world and knows nothing. Your task is to teach it and use its results to solve real-world problems.

It's as simple as that!!

Supervised Learning

We teach our computer by giving it examples. For example in the given image we are telling our system that something which is round and has red color and is eatable is called an 'Apple'. The machine understands it and next time when we ask it is able to classify the object as an 'Apple'!

Unsupervised Learning

We provide the mixed-up data to our system and it itself recognizes the differences between the data points and classify one kind of data with other.

Reinforcement Learning

It works on 'Environment-Reward-Action' model. The systems acts on the environment and gets the reward and acts on the environment again to improve its reward. The cycle continues in this way!
